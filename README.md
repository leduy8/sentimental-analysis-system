# Project Template

## Requirements

- Python 3.8+
- SQLite, MySQL or PostgreSQL

## Installation

### Set up virtual environment

```shell
python -m venv env
./venv/Scripts/activate
```

### Install dependencies

```shell
pip install -r requirements.txt
```

### Install `pre-commit` hooks

- Install `pre-commit`: https://pre-commit.com/
- Install `pre-commit` hooks:

  ```shell
  pre-commit install
  ```

## Running

```shell
python manage.py
```
